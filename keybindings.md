## Navigation

| Shortcut | What it does |
|---|---|
| `Ctrl + P` | Quick file open. Start typing a filename and it finds it instantly. Faster than clicking through folders. |
| `Ctrl + Shift + P` | Command palette. The single most powerful shortcut. If you forget everything else, remember this one. |
| `Ctrl + G` | Jump to a specific line number. |
| `Ctrl + Tab` | Cycle through open editor tabs. |
| `Ctrl + \` | Split the editor into side-by-side panes. |
| `Ctrl + B` | Toggle the sidebar on/off. More screen space when you need it. |
| `Ctrl + J` | Toggle the bottom terminal panel. |
| `Ctrl + Shift + E` | Focus the file explorer. |

| `Ctrl + Shift + L` | Select ALL occurrences of the current selection at once. Rename a variable in 0.5 seconds. |

| `Ctrl + L` | Select the entire current line. |

## Search & Replace

| Shortcut | What it does |
|---|---|
| `Ctrl + F` | Find in current file. |
| `Ctrl + H` | Find and replace in current file. |
| `Ctrl + Shift + F` | Search across ALL files in the project. |
| `Ctrl + Shift + H` | Find and replace across ALL files. |

## Terminal

| Shortcut | What it does |
|---|---|
| `` Ctrl + ` `` | Toggle the integrated terminal. |
| `Ctrl + Shift + [backtick]` | Create a new terminal instance. |
| `Ctrl + Shift + C` | Open an external terminal at the workspace root. |

## Git (with GitLens)

| Shortcut | What it does |
|---|---|
| `Ctrl + Shift + G` | Open the Source Control panel. |
| `Ctrl + Enter` | Commit staged changes (when in the Source Control panel). |

## Debugging

| Shortcut | What it does |
|---|---|
| `F5` | Start debugging (uses launch.json config). |
| `F9` | Toggle breakpoint on the current line. |
| `F10` | Step over (execute current line, move to next). |
| `F11` | Step into (dive into a function call). |
| `Shift + F11` | Step out (finish current function, return to caller). |
